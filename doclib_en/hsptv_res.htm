HSPTV Folder Material List

About the Materials

The "HSPTV Folder Materials" are image and sound data files that can be freely used when presenting your work. They can be used in works other than HSP with a simple license.
In addition, they can be freely referenced by the "WebDish service", which allows you to publish HSP scripts on the Web. We hope that many works will be created by utilizing the "HSPTV Folder Materials" and "WebDish Materials". 

		HSP3.6 WebDish Material Data
		https://dev.onionsoft.net/seed/webdish_res.html

Referencing Materials

You can easily reference materials from HSP scripts. 

			picload "bg01.jpg"

The file with the same name in the HSPTV folder will be referenced. (You can also describe the file name like "picload dir_tv+"bg01.jpg"", putting the macro "dir_tv" indicating the HSPTV material folder.)
When creating and distributing an executable file, be careful to add only the necessary materials as resources with the #pack or #epack command.
From HSP3.6β3 onwards, a module is available for loading resources collectively. 

			#include "hsptv_res.as"

By writing the above description at the beginning of the script, the following commands will be added. 

		hsptv_se _p1		Load sound effects collectively
			_p1 : 1=Simple sound effects (SE01〜SE15)
			_p1 : 2=Realistic sound effects (SND01〜SND19)
		(After that, you can play with mmplay SE01, etc.)

		hsptv_tamadot _p1	Load Tamane-chan 2D dot picture material
			_p1 : 0=64×64 1=32×32
		(The screen buffer ID is assigned to tamadot)
		(You can display it with ceput tamadot, number)

		hsptv_tamane _p1	Load Tamane-chan 2D standing picture material
			_p1 : 0=256×512 dots, 1〜8=512×1024 dots
		(The screen buffer ID is assigned to tamas)
		(You can display it with ceput tamas, number)

		hsptv_tamasc _p1	Load Tamane-chan uniform material
			_p1 : 1〜5(Variations)
		(The screen buffer ID is assigned to tamasc)

		hsptv_keisc _p1		Load Pro-Nama-chan uniform material
			_p1 : 1〜5(Variations)
		(The screen buffer ID is assigned to keisc)

		hsptv_tamawiz _p1	Load Tamane-chan wizard material
			_p1 : 1〜10(Variations)
		(The screen buffer ID is assigned to tamawiz)

		hsptv_keiknight _p1	Load Pro-Nama-chan knight material
			_p1 : 1〜10(Variations)
		(The screen buffer ID is assigned to keiknight)
