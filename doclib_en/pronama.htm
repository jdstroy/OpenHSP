Pro-nama-chan Collaboration Materials

Introduction

Pro-nama-chan materials are materials created through a collaboration project between the Programming Broadcast character "Pro-nama-chan" and the HSP3 mascot character "Tamane-chan". Please enjoy the co-starring of Tamane-chan and Pro-nama-chan.



2D Material Data Details

The Tamane-chan and Pro-nama-chan 2D materials are included in the "sample/pronama2d" and "hsptv" folders within the HSP installation folder.
PNG format files are provided. The background is filled with the standard HSP transparent color (black). In addition, alpha channel values that make it transparent in HSP3Dish are included.

"mini_kei.png" and "mini_tamane.png" are the character images for "Mini Mini Kei-chan" and "Mini Mini Tamane-chan". A total of 15 types (+ character shadow) are arranged: fantasy-style battle poses (5 types of expressions x 2) and normal clothes poses (5 types of expressions) drawn in 256 x 256 dots.








The "sample/pronama2d" folder also contains larger size images. There are 15 variations each for Tamane-chan and Pro-nama-chan, with one file each.
"mini_chr.hsp" is a sample using the 256 x 256 dot images in the "HSPTV" folder. Also, "big_chr.hsp" is a sample using larger images.
It can be used in various situations depending on your ideas. A sample script for displaying in HSP3 is also provided, so please use it in your own programs and games.
