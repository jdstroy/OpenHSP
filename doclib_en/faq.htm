Problems During Program Execution

An unknown error occurs

The error code reveals the cause of the error. The location of the error can also be estimated from the line number.

  • →Error Messages
  • →HSP3 FAQ: I don't understand the meaning and countermeasures for error messages (HSPWiki)
There are no problems when launched from the script editor, but an internal error occurs when it is made into an application

Copy the DLL and image files to the same directory as the created application.

  • →HSP Extended Plug-ins/Modules
The script posted on the website doesn't work

Modifications may be required when running a script for HSP2 with HSP3. Similarly, modifications may be required when running a script for HSP3 with HSP2. Check the version that the script posted on the website works with.

  • →Rewriting Executable File Icons
Can I distribute the DLL with my own program?

Other

The screen flickers

Try using the redraw command. At this time, do not use the cls command to clear the screen.

  • →Screen and Window
Sometimes, when pressing multiple keys on the keyboard at the same time, they are not recognized.

Most keyboards can only recognize a limited number of keys at the same time. This is a problem that cannot be avoided by software.
