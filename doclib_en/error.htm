HSP3 Error Message List

When HSP finds an error during execution due to script writing mistakes or specification errors, an error code and error line number will be displayed.

This document is for use in investigating the cause when an error occurs. Please also refer to the following site.

HSP3 FAQ: I don't understand the meaning and how to deal with error messages (HSPWiKi)

The error numbers and their contents are as follows:

1 "A system error has occurred"

This error is usually not displayed. It is displayed when a file is corrupted or an unexpected situation occurs.

2 "Syntax is incorrect"

Displayed when the description method of functions and commands is incorrect, or when variable names or label names are incorrect.

3 "Parameter value is abnormal"

Displayed when the parameter value exceeds the allowable range.

4 "An error occurred in the calculation formula"

Displayed when the calculation formula is written incorrectly or when trying to use unusable symbols or operations.

5 "Parameter omission is not allowed"

Displayed when a part where parameter omission is not allowed is omitted.

6 "Parameter type is different"

Displayed when the type specified for the parameter (string type, numeric type, etc.) is not the specified one. If a variable is specified, the variable type is incorrect.

7 "Array element is invalid"

Displayed when the value specified as an array element exceeds the number set by the dim or sdim commands.

8 "No valid label is specified"

Displayed when the parameter that should be specified as a label is incorrect, or when there is nothing after the "*" indicating the label.

9 "Subroutine or loop is nested too deeply"

Displayed when the gosub command or repeat command is executed too many times.

10 "Return outside of subroutine is invalid"

Displayed when the return command is executed without being called by the gosub command.

11 "Loop outside of repeat is invalid"

Displayed when the loop command is executed without the repeat command.

12 "File not found or invalid name"

Displayed when the specified file cannot be found or is an inaccessible file when performing file operations.

13 "Image file not found"

Displayed when the image file to be loaded by the picload command cannot be found.

14 "Error during external file call"

Displayed when a problem occurs when calling an external file with the exec command.

15 "Parentheses are incorrectly described in the calculation formula"

Displayed when the format of the parentheses "(, )" used in the calculation formula is incorrect, or the correspondence is strange.

16 "Too many parameters"

Displayed when there are too many parameters specified for the command.

17 "Exceeded the number of characters that can be handled in a string expression"

Displayed when the allowable amount for handling strings is exceeded.

18 "Specifying a variable name that cannot be assigned to"

Displayed when trying to assign to a system variable or reserved keyword.

19 "Divided by 0"

Displayed when trying to divide by 0.

20 "Buffer overflow occurred"

Displayed when the size secured by the sdim command or dim command for a string variable is exceeded, or when trying to access beyond the variable buffer.

21 "Selected an unsupported feature"

Displayed when settings are made that are not supported as functions, such as when trying to perform semi-transparent copying in palette mode with the gcopy command.

22 "Calculation formula has too many parentheses"

Displayed when the calculation formula is too complex and a stack overflow occurs during evaluation, or when the calculation cannot be performed correctly for some reason. Normally, it will not be displayed. If this is displayed, it indicates that a fatal error has occurred in the system.

23 "No variable name specified"

Displayed when an incorrect description is written in a place where a variable name should be specified as a parameter.

24 "Non-integer value specified"

Displayed when another type is detected in a place where only an integer should be specified as a parameter.

25 "Array element format is incorrect"

Displayed when the way to write array elements is incorrect, or when array elements are written in a place where they should not be written.

26 "Could not allocate memory"

Displayed when the system cannot allocate physical memory. If this is displayed, it is a fatal error.

27 "Type initialization failed"

Displayed when initialization of standard types and extended types by plugins fails. If this is displayed, it indicates that there is a fatal error in the system.

28 "No arguments set for function"

Displayed when function parameters are not written correctly.

29 "Stack overflow"

Displayed when the stack is destroyed during subroutine calls or expression evaluation. Normally, it will not be displayed. If this is displayed, it indicates that a fatal error has occurred in the system.

30 "Invalid name specified for parameter"

Displayed when an invalid name is used as a parameter. It may also be displayed if a non-existent alias or local variable is referenced.

31 "Assigned to an array variable with a different type"

Displayed when trying to assign a value of a different type than the initialized type to an array.

32 "Invalid function parameter description"

Displayed when the function parameter format is incorrect or there are not enough parentheses.

33 "Too many objects"

Displayed when the number of objects placed in a window exceeds the set limit.

34 "Type cannot be used as array/function"

Displayed when an array specification is made with a type that does not support arrays, or when trying to use a keyword that is not a function as a function.

35 "No module variable specified"

Displayed when incorrect content is written in a place where a module variable should be specified as a parameter.

36 "Module variable specification is invalid"

Displayed when the module variable specified as a parameter is initialized with the wrong module. Normally, it will not be displayed. If this is displayed, it indicates that a fatal error has occurred in the system.

37 "Failed to convert variable type"

Displayed when the type of a variable or value could not be converted or is not supported.

38 "Failed to call external DLL"

Displayed when the attempt to retrieve the address for the function call to an external DLL fails.

39 "Failed to call external object"

Displayed when an error occurs while calling a COM object.

40 "No return value set for function."

Displayed when the return command does not set a return value in a script defined as a user function.

41 "Describing a function as a command."

Displayed when a keyword defined as a function is described in the format of a command. The command name may have been misspelled, or a name that has been functionalized from HSP2 may have been mistakenly used.

42 "Command cannot be used within callback"

Displayed when a command that cannot be used is described while executing a callback routine such as an interrupt or termination process. Please see the programming manual for details on callback routines.
